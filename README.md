# Weather GraphQL Query

Basic Query Consult:
access [Weather GraphQL](https://weather-graphql-hrfpgyvnaz.now.sh/)
```
query {
  weatherByCity(city:"Cuiaba") {
    temp
    date
    time
    condition_code
    description
    currently
    cid
    city
    img_id
    humidity
    wind_speedy
    sunrise
    sunset
    condition_slug
    city_name
  }
}
```
